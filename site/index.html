Tally
Install GitHub

An over-optimized cloc clone.

Tally counts files, lines, blanks, comments, and code. It walks a tree, guesses the language, prints a table, and stops. 

$ tally .
Language Files Lines Blank Comment Code
-------- ----- ----- ----- ------- ----
Rust         6  1235   184       4 1047
TOML         9   972   146       2  824
Text         1   200    73       0  127
-------- ----- ----- ----- ------- ----
Total       16  2407   403       6 1998

Install

curl -fsSL https://jafupy.github.io/tally/install.sh | sh
tally .

Builds from source, installs to ~/.local/bin, and removes the temporary checkout.
